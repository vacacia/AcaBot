# Wave 2: Integration + Deletion

**Phase:** 02 — Plugin Reconciler
**Wave:** 2 of 3
**Requirements:** PLUG-08, PLUG-09, PLUG-10, PLUG-12, PLUG-13

## Goal

将 Wave 1 新建的六个模块接入 bootstrap/app/pipeline/control plane, 删除旧 `plugin_manager.py` 和三个 legacy 插件, 替换 HTTP API 端点. Wave 结束后 bot 使用新插件体系启动运行.

## Prerequisites

- Wave 1 全部完成: 六个新模块 + sample 插件 + 测试通过
- 现有 bot 可正常启动 (新旧共存阶段验证)

### T2.6 — 修改 `control_plane.py` [PLUG-10, PLUG-12]

### T2.8 — 修改 `http_api.py` [PLUG-10]

2. **新增端点** (5 个):
   - `GET /api/system/plugins` → `control_plane.list_plugins()` → JSON `{"plugins": [...]}`

   - `POST /api/system/plugins/reconcile` → `control_plane.reconcile_all_plugins()` → JSON `{"plugins": [...]}`
